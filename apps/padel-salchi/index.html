Reservas Padel Salchi

Private reservation system for a community padel court.

This page is a visual preview of an internal web application by Menvic. It helps residents reserve court time, manage PIN access, and handle simple administration tasks from a clear browser interface.

Project preview

Private reservation system for the La Salchi community padel court.

The live app is not linked from this page.

The screenshots show the booking flow, PIN management, administration tools, and reports.

What it includes

A simple interface focused on reservation flow, privacy, and lightweight administration.

Reservation flow

Residents can select an available day and time slot, then confirm the reservation with their portal, vivienda, and PIN.

Cancellation and slot status

The system distinguishes available, selected, and reserved slots, and supports reservation cancellation.

PIN management

Users can update their vivienda PIN from a dedicated page designed for first access or later changes.

Admin and reports

An internal administration area supports PIN reset, cleanup actions, and simple reservation reporting.
