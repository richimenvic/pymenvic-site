Notes / guides / workflows

Practical resources for tools and production workflows.

The learning area collects concise notes for Revit production, BIM documentation, pyRevit automation, app support, and small static web projects.

Resources here should be useful without becoming a course platform. The goal is reference material that helps users understand how a tool or workflow is intended to be used.

Format

Short, practical resources. No accounts, checkout, paid course area, or hidden learning portal.

Workflows Step-by-step notes for recurring production tasks.
References Checklists, examples, and implementation notes.

Resource areas

The learning route is structured around practical lookup: what the note covers, who it helps, and which public route it relates to.

Revit workflows Production notes for Revit-based architecture work. Related /revit-tools/ BIM documentation Coordination, auditing, documentation, and standards references. Related /revit-tools/ pyRevit automation Examples and patterns for automating repetitive Revit tasks. Related /revit-tools/ App notes Support page patterns, static site notes, and app documentation details. Related /apps/

Publishing approach

Learning content should be short enough to maintain and specific enough to help someone complete a task or understand a decision.

Guides A guide should describe a real workflow, not a broad topic category.
Notes A note can capture a technical decision, limitation, naming rule, or checklist.
Examples An example should connect to a tool, route, or support scenario already present on the site.
