What to include

Precise support requests are easier to answer. Add the relevant context when contacting Menvic.

For app questions App name, device model, OS version if known, and the screen or action involved.
For Revit tool questions Revit version, pyRevit context, tool name if known, and a short description of the workflow.
For privacy questions Name the app or page, then describe the request clearly. Do not include unnecessary sensitive data.
